Referências de ponto de extremidade
Esta página fornece informações de referência para os pontos de extremidade do WhatsApp.

Principais nós raiz
Nó	Descrição
/PHONE_NUMBER_ID

Representa um número de telefone específico. Este ponto de extremidade é usado para configurar a autenticação de dois fatores.

/MEDIA_ID

Representa uma objeto de mídia específico. Use este ponto de extremidade para obter e excluir objetos de mídia.

/MEDIA_URL

Representa um URL de mídia específica. Use este ponto de extremidade para baixar uma mídia de um URL.

ID do número de telefone
O ponto de extremidade /PHONE_NUMBER_ID tem as seguintes bordas:

Borda	Descrição
/PHONE_NUMBER_ID/business_compliance_info

Obtenha ou atualize as informações de conformidade de um número de telefone comercial do WhatsApp na Índia.

/PHONE_NUMBER_ID/deregister

Usada para excluir o registro de um número de telefone.

/PHONE_NUMBER_ID/media

Usada para carregar a mídia.

/PHONE_NUMBER_ID/messages

Usada para enviar mensagens.

/PHONE_NUMBER_ID/register

Usada para registrar um número de telefone.

/PHONE_NUMBER_ID/request_code

Usada para solicitar um código e verificar a propriedade de um número de telefone.

/PHONE_NUMBER_ID/verify_code

Usada para verificar a propriedade de um número de telefone.

/PHONE_NUMBER_ID/whatsapp_business_profile

Usada para obter e atualizar um perfil comercial.
