# WhatsApp Flows

WhatsApp Flows é uma forma de criar interações estruturadas para mensagens comerciais. Com Flows, empresas podem **definir, configurar e personalizar mensagens com interações ricas**, oferecendo aos clientes mais organização na comunicação.

Você pode usar Flows para:
- Gerar leads
- Recomendar produtos
- Obter novos contatos de vendas
- Qualquer outro caso onde a comunicação estruturada seja mais natural ou confortável para seus clientes.

---

## Casos de Uso Populares

Abaixo estão os casos de uso mais comuns, acompanhados de **templates**, **guias passo a passo** e **código** para facilitar a implementação. É possível criar muitos outros casos com Flows.

---

### 1. **Obter Leads para Empréstimos Pré-Aprovados**
Este guia, voltado para empresas do setor financeiro, ensina a criar uma experiência de empréstimo pré-aprovado que simplifica o processo de contratação, permitindo que usuários obtenham fundos com apenas alguns cliques.

O Flow deve:
- Apresentar um valor de empréstimo ao usuário, permitindo personalização de termos como prazo de pagamento.
- Permitir que o usuário selecione método de pagamento e forma de recebimento.
- Incluir autenticação para confirmar identidade.

Este guia pode ser adaptado para:
- Ofertas de cartão de crédito
- Calculadoras de limite de crédito
- Empréstimos para compras
- Qualificação completa para empréstimos

---

### 2. **Obter Cotação de Seguro**
Este guia ajuda a criar um Flow que permite aos usuários **personalizar e contratar seguros diretamente pelo WhatsApp**.

O Flow deve:
- Coletar dados e preferências do usuário (número de membros, riscos, prêmio, etc.).
- Permitir visualizar, personalizar e escolher planos.
- Gerar uma cotação baseada nas opções selecionadas.
- Permitir escolha da frequência de pagamento.

Pode ser adaptado para:
- Renovação de seguro
- Completar perfil
- Onboarding de clientes

---

### 3. **Coletar Intenção de Compra**
Este guia ajuda a criar um Flow para entender quais produtos ou serviços interessam aos clientes, permitindo campanhas de marketing mais assertivas.

O Flow deve:
- Coletar dados pessoais relevantes (nome, e-mail).
- Permitir que o usuário selecione produtos ou serviços de interesse.

Pode ser adaptado para qualquer caso onde seja necessário coletar informações do usuário.

---

### 4. **Oferta Personalizada**
Este guia explica como criar um Flow que oferece **ofertas personalizadas** aos clientes.

O Flow deve:
- Permitir que usuários selecionem produtos de interesse e orçamento.
- Gerar recomendações específicas.
- Enviar instruções para concluir a compra.

Ideal para eventos promocionais e engajamento.

---

## Observações Importantes
- É necessário seguir as **regulamentações locais** ao lidar com dados de usuários.
